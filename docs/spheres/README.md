# Sphere preview videos

Drop a short (5–8s) looping `.mp4` here for each sphere tile on the home page.
They autoplay muted and loop — same as the Origin section videos.

Required filenames (exact, lowercase):

| File                  | Tile           |
|-----------------------|----------------|
| `orbital.mp4`         | Orbital        |
| `ethereal-bloom.mp4`  | Ethereal Bloom |
| `nexus.mp4`           | Nexus          |
| `nebula.mp4`          | Nebula         |
| `vortex.mp4`          | Vortex         |
| `galaxy.mp4`          | Galaxy         |

Tips:
- Tile area is ~430×170 (videos are `object-fit: cover`, so any aspect ratio works — it crops to fill).
- Keep them small (a few hundred KB each) — H.264 mp4, no audio needed.
- A missing file just shows an empty dark tile; nothing breaks.
